# P1-03 — 强化序章 System Prompt 防编造规则

Status: done

## 问题

当前序章 system prompt 仅为 `"你是故事旁白。"` — 无任何防编造约束。

缺失的关键约束：
- ❌ "不要编造未发生的事件"
- ❌ "不要透露 NPC 秘密或魔女动机"
- ❌ "不要赋予 NPC 档案中没有的能力或物品"
- ❌ "严格区分玩家已知和未知信息"
- ❌ "不要使用 NPC 真名（除非 NPC 自己说出）"
- ❌ "所有描写必须基于【当前房间设施】"
- ❌ "不要改变已建立的事实"

## 修复方案

重写 `_pgm()` 返回的 system prompt，从一行扩展为结构化的旁白规则集：

```python
def _pgm(self) -> str:
    return """你是故事旁白。严格遵守以下规则：
1. 只描述玩家能直接看到或听到的内容，不做全知视角叙述。
2. 不要编造未发生的事件。如果无事发生，描述环境氛围即可。
3. 不要透露任何 NPC 的秘密、动机或内心想法。
4. 不要赋予 NPC 其档案中没有的能力、物品或关系。
5. 用外貌特征称呼陌生 NPC（如"银发少女"），只有当 NPC 自己说出名字或玩家已知时才使用真名。
6. 所有描写必须严格基于【当前房间设施】。
7. 不要引入超自然元素（除非场景设定明确允许）。
8. 叙事简洁，每次 3-5 句即可。"""
```
