# Astral

AI 主持的魔法少女互动小说引擎 · 12 NPC 并发模拟 · 3 个剧本 · 47 个 API

---

## 快速开始

1. 双击 `启动.bat`
2. 浏览器自动打开 http://127.0.0.1:8640
3. 首次使用会自动弹出 API 设置 → 添加一个 API 配置 → 测试连通性
4. 选择故事场景
5. 创建你的角色（或使用已有角色卡）
6. 选择难度模式
7. 开始游戏

## 操作指南

| 操作 | 方法 |
|------|------|
| 推进剧情 | 点击底栏"推进"按钮，或选择一个行动选项 |
| 与角色交谈 | 点击左侧角色卡片（绿色边框 = 同房间） |
| 探索/调查 | 点击选项栏中的探索/调查按钮 |
| 自由行动 | 底栏输入框输入任何文字 |
| 存档管理 | 顶栏 💾 按钮 |
| 跳过时间 | 顶栏 ⏩ 按钮 |
| 新游戏 | 顶栏 ▶ 新游戏 |

## 局域网其他设备访问

1. 命令行 `ipconfig` 获取本机 IPv4（如 `192.168.1.100`）
2. Win+R → `wf.msc` → 入站规则 → 新建规则，放行端口 8640（TCP）
3. 同一 WiFi 下，其他设备打开 `http://192.168.1.100:8640`

## 三个故事

| 故事 | 舞台 | 角色数 |
|------|------|--------|
| 天际迷宫 | 3 层神秘迷宫 | 12 人 |
| 云端假期 | 4 层天空酒店 | 12 人 |

## 难度模式

| 模式 | 体验 |
|------|------|
| 剧情模式 | 温馨日常，无忧无虑 |
| 正常模式 | 事件发生，推理为王 |
| 魔女模式 | 高难度高压挑战 |

## 角色卡

自建角色或使用预设。在游戏内编辑器或 `cards/` 目录创建 `.txt` 文件：

```
name: 角色名
age: 16
appearance: 外貌描述
magic: 魔法能力描述
personality: 性格描述（可选）
```

## API 配置

配置文件：`config_profiles.json`。支持多配置（DeepSeek / OpenAI / Gemini 等），一键切换。

## 存档

自动存档 + 手动槽位。存档文件在 `saves/` 目录。

## Debug

运行后 `logs/` 目录自动生成环形缓冲日志（每文件最多 5000 行）：

| 文件 | 内容 |
|------|------|
| `requests.log` | API 请求耗时 |
| `agents.log` | Agent 决策记录 |
| `errors.log` | 全局异常堆栈 |
| `thread_errors.log` | 线程异常 |

## 技术栈

- 后端：Flask + SSE 流式推送 + ThreadPoolExecutor
- 前端：Vanilla HTML/CSS/JS + EventSource
- LLM：OpenAI 兼容 API（config_profiles.json 管理）
- 存档：JSON
